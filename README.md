# 🚗 AutoFinder – Concessionaria Auto

AutoFinder è un'applicazione web completa per la visualizzazione e la gestione dell'inventario di una concessionaria auto.  
Offre un catalogo pubblico di auto e un'area privata per gli amministratori, dove è possibile aggiungere o rimuovere veicoli.

Realizzato con **React** (frontend) e **Node.js + Express** (backend), utilizzando un semplice file JSON come database.
